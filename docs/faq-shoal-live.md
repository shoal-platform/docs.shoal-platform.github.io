# What is `eu1.shoal.live`?

When you deploy on Shoal, your app is given a subdomain on `eu1.shoal.live` - for example, `my-app.eu1.shoal.live`. This is Shoal's shared domain for the EU1 region. It's live immediately after deploy, no DNS setup required.

You can use this address directly, or point your own custom domain at it - see [How do I set up a custom domain?](faq-custom-domain.md).
